AnyAgent Bridge

내 컴퓨터의 터미널과 Claude Code, Codex 같은 CLI 에이전트를 폰 브라우저에서 그대로 조작합니다. 새 세션은 바로 시작할 수 있고, 모바일 입력 독과 알림 모드까지 갖춘 실제 작업용 리모컨입니다.

Mobile-first control

처음 열어도 빈 터미널 앞에서 멈추지 않습니다.

새 세션에는 starter가 뜨고, agent 신뢰 프롬프트가 나오면 Enter, Esc, Ctrl-C가 엄지 위치에 남습니다.

1

QR로 폰 연결

Connect 화면에서 QR을 스캔하면 같은 세션을 폰에서 이어받습니다.

2

Starter에서 바로 실행

Start agent, `pwd`, `ls`, `git status`를 터미널 위에서 한 번에 보냅니다.

3

신뢰 프롬프트도 폰 친화적

Claude/Codex 시작 직후 Enter, Esc, 방향키, Ctrl-C가 compact launch assist에 표시됩니다.

4

조용한 알림 유지

진행 상황, 완료, 질문 알림을 기기별 모드로 조절해 방해를 줄입니다.

Built for real sessions

터미널 하나가 아니라 원격 작업 환경 전체를 다룹니다.

PTY, 파일, 프로젝트, secrets, 세션 전환, reconnect, push까지 한 화면 안에서 이어집니다.

실제 PTY 터미널

브라우저에서 내 컴퓨터의 shell을 그대로 제어하고 scrollback을 유지합니다.

Agent launcher

Claude Code, Codex, aider 등 등록한 CLI 에이전트를 한 번에 실행합니다.

모바일 입력 독

메시지 입력, 이미지 경로 첨부, 특수키, 빠른 명령, ↑↓ 명령 히스토리를 폰 화면에 맞춥니다. 멀티라인 입력에서도 방향키가 작성 중인 초안을 지우지 않습니다.

세션 매니저

검색, 이름 변경, 최근 활동, 안전한 close 확인으로 여러 세션을 관리합니다.

Files와 Markdown

허용 경로 안의 파일을 탐색하고 편집하며 preview fallback까지 제공합니다.

Projects와 Secrets

폴더를 골라 project-scoped terminal을 만들고 `.env.local`을 UI에서 관리합니다.

Exposure badge

Local, Network, Tunnel 상태를 상단에서 바로 확인해 링크 범위를 판단합니다.

Push controls

서비스 워커가 알림 모드를 적용해 progress/done/question 소음을 조절합니다.

Actual UI

문서용 목업이 아니라 실제 앱 화면입니다.

로컬 브릿지를 띄운 뒤 같은 UI가 데스크톱과 모바일 브라우저에서 동작합니다.

Security posture

기본은 localhost, 외부 공개는 명시적으로만.

브릿지는 클라우드 중개 서비스가 아니라 사용자의 컴퓨터에서 실행되는 self-hosted 도구입니다.

기본 보호

  • 처음 실행은 `127.0.0.1`에만 바인딩됩니다.
  • 접속 토큰은 로컬 `.data/auth.json`에 저장됩니다.
  • OAuth, TOTP 2FA, signed session은 선택적으로 켤 수 있습니다.

공개 전 확인

  • Tunnel은 사용자가 선택한 CLI provider를 통해서만 열립니다.
  • 상단 exposure badge로 Local, Network, Tunnel 상태를 확인합니다.
  • 공개 URL을 공유하기 전에 `docs/SECURITY.md`를 읽는 흐름을 권장합니다.
Start in one minute

Node 18+만 있으면 바로 실행합니다.

설치는 npm 패키지, 소스 체크아웃, Docker 중 원하는 방식으로 시작할 수 있습니다.

처음 쓰는 사람

환경 체크와 모바일 연결 안내가 포함된 setup wizard를 실행합니다.

npx anyagent-bridge setup
